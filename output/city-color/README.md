## 🛠️ 工作流（5 步法 + 三元映射）

```
Step 1️⃣ 资料收集（自然/人文/产业/现状/上位规划）
   ↓
Step 2️⃣ 现状采集（实地或远程图片库）
   ↓
Step 3️⃣ 色彩聚类（K-means 取主导色）
   ↓
Step 4️⃣ 三元映射
   - 自然 → 直接采样
   - 人文 → 文化转译（意象 → 物质载体 → 颜色）
   - 产业 → 意象提炼
   ↓
Step 5️⃣ 综合定调（5 色，4 沉静 + 1 点睛）
   ↓
输出：色卡（NCS/HEX）+ 设计示例 + 导则文本
```
