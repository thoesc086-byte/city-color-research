# 荔湾·西关 城市色彩提取项目

> **范围**：荔湾区西关历史文化片区（约 12 km²）
> **核心区域**：恩宁路 / 永庆坊 / 沙面 / 荔枝湾 / 陈家祠 / 泮塘五约
> **方法**：完全在线（95% 远程）
> **目标**：输出 5 个特征色 + 设计导则 + 可视化报告
> **预估工时**：2-3 周

## 🚀 工作流（4 个 Phase）

### Phase 1：资料调研（Day 1-3）
- [ ] 列代表元素清单（按色彩研究价值排序）
- [ ] 整理资料源（永庆坊改造档案、沙面建筑普查、岭南建筑研究）
- [ ] 笔记文化基因（满洲窗起源、骑楼背景、岭南民居特点）

### Phase 2：图像采集（Day 4-7）
- [ ] 网络爬取 200-500 张代表性照片
  - 民居类 80 张
  - 历史建筑 60 张
  - 自然 40 张
  - 老字号 40 张
  - 永庆坊更新 80 张
- [ ] 标注每张图（位置/类型/拍摄季节）

### Phase 3：色彩分析（Day 8-12）
- [ ] 每张图取色（colorthief / K-means）
- [ ] 全集聚类输出 30 个高频色簇
- [ ] 按类型分组分析
- [ ] 输出色彩分布热力图

### Phase 4：5 色定调 + 输出（Day 13-21）
- [ ] 三元映射（自然/人文/产业）
- [ ] 综合到 5 色（NCS + HEX）
- [ ] 输出色卡（PNG/SVG/Figma）
- [ ] 写报告（含设计示例）
