# 候选区域评估 · 哪个城市的哪个区方便我们提取产出

> 评估维度：基因清晰度 / 管理边界 / 建设阶段 / 资料可得性 / 视觉特色 / 落地难度

## 🎯 推荐序列（综合评分排序）

### 🥇 Tier 1：最容易出活（强推）

#### 1. 深圳·前海合作区（120 km²）
**得分：⭐⭐⭐⭐⭐ 9.5/10**

| 维度 | 评估 |
|---|---|
| 自然基因 | ✅ 滨海 + 红树林湿地 + 珠江入海口 |
| 人文基因 | ✅ "敢闯敢试"特区精神 + 香港风貌借鉴 |
| 产业基因 | ✅ 现代服务业 + 金融 + 跨境贸易 |
| 管理边界 | ✅ 前海管理局独立运作 |
| 建设阶段 | ✅ 70% 在建，色彩干预正当时 |
| 资料 | ✅ 公开资料丰富，可走访 |
| 视觉差异 | ✅ 与深圳其他区色彩反差大 |

**预期 5 色（推测）**：海湾蓝 / 红树绿 / 沙滩米 / 创新银 / 港式金

## 🎁 我的最终推荐：苏州工业园区

### 为什么选它？

1. **资料最容易拿到**
   - 苏州本身有完整的城市色彩研究（古城+园区双套）
   - 园区管委会公开资料丰富
   - iWiki / KM 内可能有相关资料

2. **基因纯度极高**
   - 自然：金鸡湖 + 独墅湖（地标级）
   - 人文：中新合作（独特性）
   - 产业：生物医药 + AI（清晰）

3. **教学价值最高**
   - 与苏州古城形成"古今对照"，可以延伸出**双城色彩对比**研究
   - 是国内"分区做色彩"的最佳样板

4. **离北京远 → 不重复亦庄**
   - 亦庄（北京产业新城）vs 园区（苏州产业新城）—— 同质项目跨区对比

5. **可以做出**
   - 一份 5 色色卡
   - 一份**"古城 vs 园区"对比报告**
   - 一份方法论 SOP

## 🚀 立项建议（如果选苏州工业园区）

### Phase 1：远程资料调研（1 周）
- iWiki / 公开资料 / 同类项目对比
- 输出"基因清单"

### Phase 2：色彩象征性采集（不实地，用 Google/Baidu/小红书图）
- 收集 200-500 张园区代表性照片
- 用 Python 脚本自动跑色彩聚类
- 输出"高频色簇"

### Phase 3：三元映射 + 5 色定调（1 周）
- 自然/人文/产业三路并行
- 综合压缩到 5 色

### Phase 4：可视化输出（1 周）
- 5 色色卡（PNG/SVG/Figma）
- 配套设计示例（建筑立面/标识/海报模拟）
- 报告 PDF

**总工时：3-4 周可出 demo 版**（专业版要 6-12 个月）
