# openai-image-restyle · OpenClaw Skill

使用 OpenAI GPT-Image API 把任意图片转换成"极简黑白线描"等风格。
**核心原则**：只换风格，不动构图（保持原图主体的位置、大小、比例、姿态完全不变）。

## ✨ 特性

- 🎨 内置「极简黑白线描签名风」prompt（实测 26 张图效果稳定）
- 🔒 「只换风格不动构图」硬约束（FORBIDDEN 词列表防止 AI 乱改）
- 📐 自动尺寸选择 + 可选 sips 重采样回原始尺寸
- 💰 平均每张 ~$0.06（GPT-Image-1.5 high 质量）

## 📦 安装方法

### 方法 1：作为 OpenClaw Skill（推荐）

```bash
# 解压本包后
cp -r openai-image-restyle ~/.openclaw/skills/
chmod +x ~/.openclaw/skills/openai-image-restyle/scripts/restyle.sh
# 重启 OpenClaw（或新会话即可生效）
```

### 方法 2：直接用脚本（独立使用）

```bash
chmod +x openai-image-restyle/scripts/restyle.sh
export OPENAI_API_KEY="你的 OpenAI Key"
./openai-image-restyle/scripts/restyle.sh ./input.jpg ./output.png
```

## 🔑 准备 OpenAI API Key

需要可调用 `gpt-image-1.5` / `gpt-image-2` 的 OpenAI 账号。

```bash
export OPENAI_API_KEY="sk-proj-xxx..."
```

⚠️ 本 skill **不会保存 key 到任何文件**，仅通过环境变量读取。
